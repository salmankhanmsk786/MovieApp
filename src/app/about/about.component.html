About this Application

Welcome to the Movie Management Application! This application is designed to help you manage your movie collection with ease. Below is a detailed overview of the features and functionalities provided by this application.

Features

  • User Authentication: Users can register, log in, and log out securely. Password reset functionality is also available.
  • Movie Management: Users can add new movies to their collection, edit existing movies, and delete movies they no longer want to keep.
  • Movie Reviews: Users can view reviews for each movie and add their own reviews to share their thoughts with others.
  • Real-time Chat: Users can participate in a real-time chat to discuss movies with other users. This feature is powered by SignalR for real-time communication.
  • Responsive Design: The application is designed to be responsive and works well on both desktop and mobile devices.
  • Email Notifications: The application uses the Mailtrap API to send real emails with a token link to users for password reset. This ensures that users can securely reset their passwords if they forget them.
  • Advanced Search and Filtering: The application uses complex forms and Angular forms for searching, sorting, paging, and filtering movies. This allows users to easily find and manage their movie collection based on various criteria.

Technologies Used

  • Angular: The front-end of the application is built using Angular, a powerful framework for building dynamic web applications.
  • ASP.NET Core: The back-end of the application is built using ASP.NET Core, a cross-platform framework for building modern, cloud-based, and internet-connected applications.
  • SignalR: SignalR is used for real-time communication in the chat feature, allowing users to send and receive messages instantly.
  • Bootstrap: Bootstrap is used for styling and layout, providing a responsive and visually appealing design.
  • Material Design: Angular Material components are used to enhance the user interface with modern design elements.
  • Mailtrap: Mailtrap is used to send real emails with a token link for password reset, ensuring secure email notifications.
  • Search and Filter: Use the search and filter options to find movies based on various criteria such as title, genre, and rating.
  • User-friendly Interface: The application has a super easy-to-use user interface for navigation, thanks to Angular Material. This ensures a smooth and intuitive user experience.

How to Use

  1. Register: Create a new account by providing your email and password.
  2. Log In: Log in to your account using your email and password.
  3. Reset Password: If you forget your password, use the "Forgot Password" link to receive an email with a token link to reset your password.
  4. Add Movies: Navigate to the "Movies" section and click on "Add New Movie" to add a new movie to your collection.
  5. Edit Movies: Click on the "Edit" button next to a movie to update its details.
  6. Delete Movies: Click on the "Delete" button next to a movie to remove it from your collection.
  7. View Reviews: Click on a movie to view its details and read reviews from other users.
  8. Add Reviews: Add your own review to a movie by filling out the review form and submitting it.
  9. Join the Chat: Participate in the real-time chat to discuss movies with other users.
